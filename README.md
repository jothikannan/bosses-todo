# bosses-todo
Keeps the todo list of pointy haired boss

### Major Dependencies

- Typescript
- Express
- @Overnight/Core
- @Overnight/Jwt
- Passport
- Mongoose
- Inversify
- Winston
- Jsonwebtoken


### Commands to run

- Build Code: `npm run build`
- Run Unit test cases: `npm run test`
- Run app in development mode: `npm run play`
- Run app in production mode: `npm run prod`
